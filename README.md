# Agricultural Commodity Price Forecasting System

Built an end-to-end machine learning pipeline to forecast commodity prices using advanced regression models and time-series feature engineering.

## Tech Stack
- Python
- XGBoost
- LightGBM
- Scikit-learn
- Pandas
- FastAPI
- React & Vite (Frontend Dashboard)

## Features
✅ Multi-model training with performance comparison  
✅ RMSE & R² based evaluation  
✅ Market regime detection  
✅ Interactive dashboard  

## Result
Improved forecasting reliability and enabled data-driven agricultural pricing insights.
